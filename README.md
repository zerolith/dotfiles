# dotfiles

My personal dotfiles

[Link to Dotfiles Manager - Chezmoi](https://www.chezmoi.io)

## Install on Mac
```bash
brew install chezmoi
```

## One Line installer
```bash 
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply zerolith
```
